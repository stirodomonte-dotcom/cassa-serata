# Cassa Serata — PWA

App per la gestione ingressi di serate ed eventi.

## Come pubblicare su GitHub Pages (gratis)

1. Vai su [github.com](https://github.com) e crea un account se non ce l’hai
1. Crea un nuovo repository chiamato `cassa-serata`
1. Carica tutti i file di questa cartella nel repository
1. Vai su **Settings → Pages → Source** e seleziona `main` branch
1. GitHub ti darà un link tipo `https://tuonome.github.io/cassa-serata`
1. Apri quel link dal telefono → Safari mostrerà “Aggiungi alla schermata Home”
1. L’app si installa come una vera app, senza browser visibile

## File

- `index.html` — app principale
- `manifest.json` — configurazione PWA
- `sw.js` — service worker (funzionamento offline)
- `icon-192.png` / `icon-512.png` — icone app
